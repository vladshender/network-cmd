## Стандартна перевірка

- **Перегляд стану портів на платі**:
    _Для перегляду стану всіх портів на певній платі:_
    ```bash
    display board 0/16
    display ont info 0 16 1 3 - перегляд інформації по ONU(0 - frame, 16 - board, 1 - port, 3 onu-id)
    display ont info option run-state 0 16 1 3 - перегляд оптичних показників по ONU
    ```

- **Перегляд MAC-адрес**:
    _Щоб переглянути MAC-адреси на порту або з пристрою, увійдіть в привілегійований режим:_
    ```bash
    enable
    ```
    - Перегляд MAC-адрес на порту (наприклад, порт 0/11/1):
        ```bash
        display mac-address port 0/11/1
        ```
    - Перегляд MAC-адрес на конкретному ONU (наприклад, ONU 11 на порту 0/11/1):
        ```bash
        display mac-address port 0/11/1 ont 11
        ```
    - Перегляд MAC-адрес в конкретному VLAN (наприклад, VLAN 3311):
        ```bash
        display mac-address vlan 3311
        ```

- **Перегляд активних алармів**:
    _Для перегляду активних алармів на порту або ONU абонента:_
    - Для перегляду всіх активних алармів:
        ```bash
        display alarm active all
        ```
    - Для перегляду активних алармів на певному порту або ONU (наприклад, фрейм 0, плата 2, порт 4, ONU 5):
        ```bash
        display alarm active alarmparameter 0/2/4 5
        ```
    - Для перегляду активних алармів на всіх ONU на певному порту (наприклад, фрейм 0, плата 2, порт 4):
        ```bash
        display alarm active alarmparameter 0/2/4
        ```

- **Перегляд історії алармів**:
    _Для перегляду історії алармів, що були на порту або ONU абонента:_
    - Перегляд історії всіх алармів:
        ```bash
        display alarm history all
        ```
    - Перегляд історії алармів на певному порту або ONU:
        ```bash
        display alarm history alarmparameter 0/2/4 5
        ```
    - Перегляд історії алармів по всьому порту для всіх ONU:
        ```bash
        display alarm history alarmparameter 0/2/4
        ```

## Вимкнення порту чи ONU

- **Вимкнення порту або ONU **:
- Через Систему керування
    _Для вимкнення порту або ONU, використовуйте систему управління Huawei NCE._
    - Підключіться до NCE для виконання операцій вимкнення.
    - Виберіть відповідний порт або ONU для деактивації через інтерфейс управління.
- Через обладнання
  ```bash
  enable
  config
  interface giu 0/20
  shutdown
  ```
