## Стандартна перевірка

### **Iskratel MSAN абонентська плата**  
_Плати, що мають в назві цифру від 1 до 39999_

- **Отримання інформації про порт**:  
    _Щоб отримати загальну інформацію про конкретний DSL порт (наприклад, порт 18):_
    ```bash
    show dsl port 18
    ```

- **Перегляд MAC-адрес на порту**:  
    _Для перегляду MAC-адрес на порту, де використовуються різні профілі абонентів:_
    
    - Для **DHCP профілю** (профіль 40):
    ```bash
    show bridge mactable interface dsl18:1_40
    ```
    
    - Для **PPPoE профілю** (профіль 33):
    ```bash
    show bridge mactable interface dsl18:1_33
    ```
    
    - Для **IPTV профілю** (профіль 32):
    ```bash
    show bridge mactable interface dsl18:1_32
    ```

    _Щоб переглянути всі MAC-адреси на порту:_
    ```bash
    show bridge mactable
    ```

- **Отримання загальної інформації про систему**:  
    _Для отримання загальної інформації про обладнання:_
    ```bash
    show system info
    ```

    _Для перегляду ідентифікатора обладнання (схеми, шасі):_
    ```bash
    show shelf-id
    ```

    _Для перегляду статусу/налаштувань системи:_
    ```bash
    show b p
    ```

- **Перегляд логів на обладнанні**:  
    _Щоб переглянути поточні аларми та інші логи на обладнанні:_
    ```bash
    show alarm
    ```
