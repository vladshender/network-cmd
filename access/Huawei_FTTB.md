## Стандартна перевірка

- **Перевірка стану порту**:
    _Для перегляду опису та статусу інтерфейсу:_
    ```bash
    display interface description
    display interface brief
    display interface Ethernet 0/0/5
    ```

- **Перевірка MAC-адрес на порту**:
    _Для перегляду MAC-адрес на вказаному Ethernet порту (наприклад, 0/0/5):_
    ```bash
    display mac-address Ethernet 0/0/5
    ```
    _Для перегляду MAC-адрес у конкретному VLAN (наприклад, VLAN 602):_
    ```bash
    display mac-address vlan 602
    ```

- **Перегляд логів**:
    _Для перегляду логів з фільтрацією по конкретному порту (наприклад, Ethernet0/0/5):_
    ```bash
    display logbuffer | include Ethernet0/0/5
    ```

- **Перегляд логів для СРС помилок**:
    _Для перевірки наявності СРС (Cyclic Redundancy Check) помилок на порту:_
    ```bash
    display trapbuffer | include Ethernet0/0/5
    ```

- **Перегляд оптичних показників на інтерфейсі**:
    _Для перегляду детальної інформації по оптичних показниках на інтерфейсі (наприклад, Ethernet 0/0/3):_
    ```bash
    display transceiver interface Ethernet 0/0/3 verbose
    ```

- **Перегляд поточної конфігурації**:
    _Для перегляду поточної конфігурації пристрою:_
    ```bash
    display current-configuration
    ```

## Вимкнення інтерфейсу

- **Вимкнення або увімкнення порту**:
    _Щоб адміністратору вимкнути або увімкнути порт, потрібно увійти в конфігураційний режим та виконати команди:_
    1. Увійдіть в конфігураційний режим:
        ```bash
        system-view
        ```
    2. Зайдіть в налаштування конкретного інтерфейсу (наприклад, Ethernet 0/0/5):
        ```bash
        interface Eth0/0/5
        ```
    3. Вимкніть порт адміністративно:
        ```bash
        shutdown
        ```
    4. Увімкніть порт:
        ```bash
        undo shutdown
        ```
    5. Вийдіть з конфігураційного режиму:
        ```bash
        quit
        ```
