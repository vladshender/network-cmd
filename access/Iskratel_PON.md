## Стандартна перевірка

- **Перегляд стану всіх портів та ONU**:
    _Щоб переглянути стан усіх портів та ONU на пристрої:_
    ```bash
    show port all
    ```

- **Перегляд інформації по конкретному порту**:
    _Щоб отримати детальну інформацію по конкретному порту (наприклад, порт 0/6/2):_
    ```bash
    show port 0/6/2
    ```

- **Перегляд MAC-адрес на порту**:
    _Щоб переглянути MAC-адреси, які вивчені на певному порту (наприклад, порт 0/1/2):_
    ```bash
    show mac-addr-table interface 0/1/2
    ```
    _Для перегляду MAC-адрес в конкретному VLAN (наприклад, VLAN 602):_
    ```bash
    show mac-addr-table vlan 602
    ```

- **Перегляд детальної інформації про ONU**:
    _Щоб переглянути детальну інформацію по ONU, підключеному до певного порту (наприклад, порт 0/1/2):_
    ```bash
    show onu interface 0/1/2
    ```

- **Перегляд логів на обладнанні**:
    _Щоб переглянути помилки та інші важливі логи (наприклад, файл MsgErr):_
    ```bash
    show logging file MsgErr
    ```

## Додаткова перевірка

- **Перегляд інформації про пристрій (MAC, uptime та ін.)**:
    _Для перегляду загальної інформації про пристрій:_
    ```bash
    show sysinfo
    show version
    show hardware
    show boot package
    show shelf-id
    ```

- **Перезавантаження ONU**:
    _Для перезавантаження ONU на певному інтерфейсі (наприклад, інтерфейс 0/1/2):_
    1. Увійдіть в конфігураційний режим:
        ```bash
        configure
        ```
    2. Зайдіть в налаштування конкретного інтерфейсу:
        ```bash
        interface 0/1/2
        ```
    3. Вимкніть порт:
        ```bash
        shutdown
        ```
    4. Увімкніть порт:
        ```bash
        no shutdown
        ```
    _Паралельно потрібно завершити сесію в АРМУС (система управління)._

- **Перегляд DHCP сесії по інтерфейсу**:
    _Щоб переглянути активні DHCP сесії для конкретного інтерфейсу (наприклад, інтерфейс 0/5/33):_
    ```bash
    show dhcpsnooping interface 0/5/33
    ```

- **Очистка DHCP сесії**:
    _Щоб очистити DHCP сесію для конкретного інтерфейсу (наприклад, інтерфейс 0/5/33/2):_
    ```bash
    clear dhcpsnooping interface 0/5/33/2
    ```
    _Замість `x/y/z/2` можна використовувати інтерфейс, для якого потрібно переглянути MAC-адреси або сесії._
