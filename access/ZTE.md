## Стандартна перевірка
##### Перейти в конфіг режим
    enable(при паролі просто ентер)
##### Стан інтерфейсів + оптичні показники
     show port 1/1
##### Перегляд МАС-адреси по порту
    show mac dynamic port 1/2
    show mac dynamic vlan 602
    В залежності від моделі може бути
    show fdb port 1/2
    show fdb vlan 602
##### Переглянути логи
    Часто збита дата на обладнанні і по часу логи не відповідають
    show terminal log 
## Додатково
##### Подивитись uptime, soft
    show version
##### Подивитись дату на обладнанні
    show date-time
##### Зміна швидкості на порту
    set port 1/2 speed 1000
##### Перезавантажити порт
    set port 1/2 disable
    set port 1/2 enable
