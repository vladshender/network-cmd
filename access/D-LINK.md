## Стандартна перевірка

- **Перевірка стану інтерфейсів**:
    _Для перегляду загальної інформації по всіх портах:_
    ```bash
    show ports
    ```

- **Перевірка MAC-адрес на порту**:
    _Для перегляду MAC-адрес, що з'являються на вказаному порту:_
    ```bash
    show fdb port 2
    ```

- **Перевірка MAC-адрес у VLAN**:
    _Для перегляду MAC-адрес у конкретному VLAN (наприклад, VLAN 602):_
    ```bash
    show fdb vlanid 602
    ```

- **Перевірка оптичних показників**:
    _Для перегляду оптичних показників по всіх портах:_
    ```bash
    show ddm ports status
    ```

- **Перегляд логів**:
    _Для перегляду загальних логів пристрою:_
    ```bash
    show log
    ```

## Зміни порту

- **Вимкнення порту**:
    _Для вимкнення порту (наприклад, порт 1:2):_
    ```bash
    config ports 1:2 state disable
    ```

- **Увімкнення порту**:
    _Для увімкнення порту (наприклад, порт 1:2):_
    ```bash
    config ports 1:2 state enable
    ```

- **Зміна швидкості порту**:
    _Для зміни швидкості порту (наприклад, 1000 Мбіт/с на порті 1:2):_
    ```bash
    config ports 1:2 speed 1000
    ```
