## Перевірка стану інтерфейсів

- **Подивитись статус та опис всіх портів**:
    ```bash
    show interface des
    show interface brief
    ```

- **Подивитись чи приходить MAC-адреса на порт**:
    ```bash
    show mac address-table interface Gi/XGi
    ```

- **Перевірити, чи приходять MAC-адреси у VLAN**:
    ```bash
    show mac address-table vlan X
    ```

- **Пошук логів по конкретному порту**:
    ```bash
    show log | include 1/21
    ```

## Додатково

- **Перевірити uptime обладнання**:
    ```bash
    show version
    ```
    _Це дозволить побачити час, який пройшов з останнього перезавантаження пристрою._

- **Перегляд оптичних показників**:
    ```bash
    show interface transceiver detail
    ```
    _Застосовується для перевірки стану оптичних модулів._

- **Перегляд конфігурації всього обладнання**:
    ```bash
    show running-config
    ```

- **Перевірка статусу блоків живлення, охолодження та температури**:
    ```bash
    show env all
    ```

- **Перевірити на яких портах є VLAN**:
    ```bash
    show vlan id 3301
    ```

## Робота з портами

- **Вимкнути/увімкнути порт**:
    1. Перейдіть у режим конфігурації:
        ```bash
        configure terminal
        ```
    2. Виберіть потрібний інтерфейс:
        ```bash
        interface Gi1/21
        ```
    3. Для вимкнення порту:
        ```bash
        shutdown
        ```
    4. Для увімкнення порту:
        ```bash
        no shutdown
        ```

- **Додавання VLAN на порт**:
    1. Перейдіть у режим конфігурації:
        ```bash
        conf term
        ```
    2. Виберіть інтерфейс:
        ```bash
        interface Gi0/11
        ```
