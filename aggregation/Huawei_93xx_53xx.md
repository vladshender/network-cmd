## Перегляд інформації по інтерфейсам

- **Перегляд фізичних та логічних інтерфейсів**:
    ```bash
    display interface description
    display interface GigabitEthernet 1/6/1/0
    display interface XGigabitEthernet 1/2/0/0
    display interface Eth-Trunk0
    ```

## Перевірка MAC-адрес

- **Перегляд всіх MAC-адрес на обладнанні**:
    ```bash
    display mac-address
    ```

- **Перегляд всіх MAC-адрес в VLAN**:
    ```bash
    display mac-address dynamic vlan 4092
    ```

- **Перегляд всіх MAC-адрес на конкретному порту**:
    ```bash
    display mac-address dynamic GigabitEthernet 1/1/0/0
    ```

- **Перевірка наявності конкретної MAC-адреси**:
    ```bash
    display mac-address 0013-d3c5-d3de
    ```

## Перегляд логів на обладнанні

- **Перегляд буфера логів**:
    ```bash
    display logbuffer
    ```

- **Перегляд буфера трапів (для перевірки наявності СРС на інтерфейсі)**:
    ```bash
    display trapbuffer
    ```

## Перевірка uptime та версії програмного забезпечення

- **Перевірка uptime та версії ПЗ**:
    ```bash
    display version
    ```
    _Ця команда дозволяє побачити, скільки часу працює пристрій та яка версія програмного забезпечення використовується._

## Перегляд оптичних показників

- **Перегляд оптичних показників на інтерфейсі**:
    ```bash
    display transceiver interface XGi1/1/1 verbose
    ```
    _Ця команда надасть детальну інформацію про оптичні модулі на вибраному інтерфейсі._

## Перевірка часу на пристрої

- **Перегляд актуального часу на пристрої**:
    ```bash
    display clock
    ```

## Перевірка стану лінійних та керуючих карт

- **Перегляд стану лінійних та керуючих карт**:
    ```bash
    display device
    ```

## Перезавантаження порту

- **Перезавантаження порту**:
    1. Перейдіть в режим системного перегляду:
        ```bash
        system-view
        ```
    2. Виберіть інтерфейс:
        ```bash
        interface Gi1/1/1
        ```
    3. Для вимкнення порту:
        ```bash
        shutdown
        ```
    4. Для увімкнення порту:
        ```bash
        undo shutdown
        ```
    5. Вийдіть з конфігурації:
        ```bash
        quit
        ```
