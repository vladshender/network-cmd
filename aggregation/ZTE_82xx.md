## Стандартна перевірка

- **Перегляд стану портів**:
    ```bash
    show interface description
    ```

- **Перегляд MAC-адрес**:
    - Перегляд всіх MAC-адрес:
        ```bash
        show mac table
        ```
    - Перегляд MAC-адрес на конкретному порту:
        ```bash
        show mac table interface gei-0/1/0/40
        ```
    - Перегляд MAC-адрес в конкретному VLAN:
        ```bash
        show mac table vlan 1110
        ```

- **Перегляд логів**:
    - Перегляд логів з фільтром за конкретним портом:
        ```bash
        show logging alarm | include 0/1/0/40
        ```

## Додатково

- **Перегляд оптичних показників на інтерфейсі**:
    ```bash
    show optical-inform detail info interface gei-0/1/0/40
    ```
    _Ця команда дає детальну інформацію про оптичні модулі на вибраному інтерфейсі._

- **Перегляд конфігурації**:
    - Перегляд конфігурації всього пристрою:
        ```bash
        show running-config
        ```
    - Перегляд конфігурації для конкретного інтерфейсу:
        ```bash
        show running-config-interface gei-0/1/0/40
        ```

- **Перевірка uptime та версії ПЗ**:
    ```bash
    show version
    ```
    _Ця команда виводить інформацію про час роботи пристрою та його програмне забезпечення._
